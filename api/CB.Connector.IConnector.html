    Interface IConnector

    Provides a base interface to specify the backbone of a functional connector.

    Namespace: CB.Connector
    Assembly: CB.Connector.dll
    Syntax
    public interface IConnector
    Remarks

    The IConnector intermediates the communication between the data provider and CB Server. A functional connector acts as a relational database that stores a collection of data and metadata.
