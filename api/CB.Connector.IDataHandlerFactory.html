    Interface IDataHandlerFactory

    Factory implementation that always creates IDataHandler objects.

    Namespace: CB.Connector
    Assembly: CB.Connector.dll
    Syntax
    public interface IDataHandlerFactory
    Remarks

    A class that implements the IDataHandlerFactory interface has no behavior except to manufacture new instances of classes that implement the IDataHandler interface.
