    Interface IConstraint

    Provides a base interface for defining a SQL constraint that is an assertion that indicates a restriction that must be satisfied by any valid realization of the model containing the constraint (that is, rules).

    Remarks

    An IConstraint is attached to a set of constrained elements, and it represents supplementary semantic information about those elements.
