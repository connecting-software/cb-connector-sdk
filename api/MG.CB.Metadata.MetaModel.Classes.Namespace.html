Remarks

ResolveName() Methods involves the following steps:

  1. The expected number of namespaces to include on the QualifiedName is calculated using type.
  2. Determines if the name is the QualifiedName. If so, returns the name; otherwise, the default namespaces are added to the beginning of the name.
