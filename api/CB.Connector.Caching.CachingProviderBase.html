    The object to insert.

    The fixed date and time at which the cache entry will expire.
    Remarks

    The typical behavior is an insert-or-update operation. If an item that matches key does not exist in the cache, key and value are used to insert as a new cache entry; otherwise, the cache entry is updated by using value.
