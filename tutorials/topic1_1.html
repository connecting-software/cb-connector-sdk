    Creating a new solution in Microsoft Visual Studio

    Creating a new solution Using Visual Studio Project Template

    Do the following:

    1. Install the VSIX Project Template.

      Installing the VSIX Project Template for Visual Studio will create a new Connector Class Library (.Net Framework) template in Visual Studio.

    2. Open Visual Studio and create a New Project.

    3. Select Other Languages > Visual C# > Connector Class Library (.NET Framework).

    4. Name the project.

      The recommended format for the name is CB<YourSource>Connector. For example, CBGoogleConnector.

    5. Select your preferred Location.

    6. Click OK to open New CB Connector Class Library dialog box.

    7. Fill out the options as you want and click Finish to create your project.

    Creating a new empty solution

    Do the following:

    1. Open Visual Studio and create a New Project.

    2. Select Other Languages > Visual C# > Class Library (.NET Framework).

    3. Name the project.

      The recommended format for the name is CB<YourSource>Connector. For example, CBGoogleConnector.

    4. Select your preferred Location.

    5. Click OK to create your project.
