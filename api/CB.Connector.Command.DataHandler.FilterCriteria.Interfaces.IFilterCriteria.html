    Interface IFilterCriteria

    Represents an SQL search condition that encapsulates a processing step where the input data is filtered according with a specific operation. It is used to determine how to filter the input values.

    Namespace: CB.Connector.Command.DataHandler.FilterCriteria.Interfaces
    Assembly: CB.Connector.dll

    Parent

    Declaration
    IFilterCriteria Parent { get; set; }
    Property Value
    Type Description
    IFilterCriteria

    The parent of this IFilterCriteria. If a IFilterCriteria has just been created and not yet added to the tree, or if it has been removed from the tree, the parent is null. Returns null; this IFilterCriteria do not have parent.
