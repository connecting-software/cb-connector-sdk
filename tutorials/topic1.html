    Getting Started

    Prerequisites

    The following prerequisites must be fulfilled in order to get started with CB Connector SDK:

    • Microsoft .NET Framework 4.6 or later
    • Microsoft Visual Studio 2010 or later
    • Connect Bridge Platform with a license for connector development
    • CB.Connector.dll
    Note

    To download the CB Connector SDK go to the download page and click Download file. Look for CBConnectorSDK_1.0.zip in the list of download files.

    Note

    To download the Connect Bridge Platform, go to the download page and click Download file. Look for connectbridge_41913868.exe in the list of download files.

    Getting Started with CB Connector SDK

    This guide contains a variety of documentation articles, walkthroughs and tutorials that can help you get started quickly.

    The following steps are required to create your own custom connector:

    1. Creating a new solution in Microsoft Visual Studio.
    2. Implementing mandatory classes
    3. Define your connector logic
    4. Building your solution
    5. Making your own custom connector recognizable by CB Server
    Note

    A list of samples can be found in GitHub repository.
